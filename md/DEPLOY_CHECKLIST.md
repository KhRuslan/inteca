# ✅ Deploy Checklist для INTECA

Используйте этот чеклист перед деплоем в продакшен.

---

## 🔐 Безопасность

- [ ] Изменен пароль админки с `admin123` на сложный
- [ ] `.env` файл в `.gitignore`
- [ ] Supabase RLS политики проверены
- [ ] API keys не в коде (только в .env)
- [ ] HTTPS включен на домене
- [ ] CORS настроен в Supabase

---

## 🎨 Контент

- [ ] Все тексты проверены на ошибки
- [ ] Все изображения оптимизированы (WebP, <500KB)
- [ ] Логотипы заменены на финальные
- [ ] Контактная информация актуальна
- [ ] Все ссылки работают
- [ ] Посты блога созданы (минимум 3-5)

---

## ⚡ Производительность

- [ ] `npm run build` без ошибок
- [ ] Bundle size проверен (<500KB gzipped)
- [ ] Lighthouse score >90
- [ ] Изображения сжаты
- [ ] Lazy loading включен
- [ ] CDN настроен (Supabase Storage)
