# Руководство по работе с блогом INTECA

## Страницы блога

### 1. Список постов (`/blog`)

Страница отображает все опубликованные посты с возможностью:
- **Поиска** - по заголовку и описанию
- **Фильтрации по тегам** - выбор конкретного тега
- **Избранные посты** - отображаются отдельно вверху страницы
- **Все посты** - основной список статей

### 2. Страница поста (`/blog/:id`)

Отдельная страница статьи с:
- Полным содержимым поста (поддержка HTML)
- Информацией об авторе и дате
- Тегами
- Блоком похожих статей (по тегам)

## Управление блогом через админку

### Вход в админку

1. Откройте `/admin`
2. Войдите с паролем: `admin123`
3. В боковом меню выберите **"Blog Posts"**

### Создание нового поста

1. Нажмите кнопку **"Создать пост"**
2. Заполните форму:
   - **Заголовок** - название статьи
   - **Краткое описание** - текст для превью (2-3 предложения)
   - **Контент** - полный текст статьи в HTML формате
   - **Автор** - имя автора (по умолчанию "Oleg Tsoy")
   - **Дата** - дата публикации
   - **Теги** - через запятую (например: education, business, strategy)
   - **URL изображения** - путь к обложке (например: /hero-section.png)
   - **Избранный пост** - чекбокс для featured постов
3. Нажмите **"Сохранить"**

### Форматирование контента

Контент поста поддерживает HTML. Примеры:

```html
<h2>Заголовок второго уровня</h2>
<p>Обычный параграф текста.</p>

<h3>Подзаголовок</h3>
<p>Еще текст с <strong>жирным</strong> и <em>курсивом</em>.</p>

<ul>
  <li>Элемент списка 1</li>
  <li>Элемент списка 2</li>
</ul>

<ol>
  <li>Нумерованный список</li>
  <li>Второй пункт</li>
</ol>

<blockquote>
  <p>Цитата или важное замечание</p>
</blockquote>
```

### Редактирование поста

1. В списке постов найдите нужный
2. Нажмите кнопку **"Изменить"**
3. Отредактируйте поля
4. Нажмите **"Сохранить"**

### Удаление поста

1. В списке постов найдите ненужный пост
2. Нажмите кнопку **"Удалить"**
3. Подтвердите удаление

## Работа с тегами

### Что такое теги?

Теги помогают категоризировать посты и позволяют пользователям фильтровать статьи по темам.

### Рекомендуемые теги:

- `education` - образование
- `business` - бизнес
- `strategy` - стратегия
- `case-method` - кейс-метод
- `harvard` - Harvard методология
- `corporate` - корпоративное обучение
- `universities` - университеты
- `students` - студенты
- `leadership` - лидерство
- `management` - менеджмент

### Как добавить теги:

В поле "Теги" введите их через запятую:
```
education, case-method, harvard
```

## Избранные посты (Featured)

Featured посты:
- Отображаются отдельным блоком вверху страницы блога
- Имеют более крупное отображение
- Отмечены красным бейджем "Featured"

Рекомендуется отмечать как Featured:
- Самые важные и актуальные статьи
- Анонсы новых программ
- Статьи с высокой вовлеченностью

## Работа с изображениями

### Добавление изображения к посту:

1. Загрузите изображение в папку `/public/`
2. В форме поста укажите путь: `/название-изображения.jpg`
3. Поддерживаемые форматы: JPG, PNG, SVG

### Рекомендуемые размеры:

- **Обложка поста**: 1200x600px или 2:1 соотношение
- **Формат**: JPG или PNG
- **Размер файла**: не более 500KB для быстрой загрузки

## Советы по написанию постов

### Заголовок:
- Краткий и информативный
- 5-10 слов
- Отражает суть статьи

### Краткое описание:
- 2-3 предложения
- Краткая суть статьи
- Мотивирует прочитать полностью

### Контент:
- Структурированный (используйте заголовки H2, H3)
- С примерами и списками
- Легко читаемый
- 500-1500 слов

### SEO рекомендации:
- Используйте релевантные теги
- Хорошие заголовки
- Качественные изображения
- Регулярно обновляйте контент

## Хранение данных

- Все посты хранятся в **localStorage** браузера
- Данные сохраняются автоматически
- При очистке localStorage посты вернутся к дефолтным

## Резервное копирование

Чтобы сохранить посты:
1. Откройте DevTools (F12)
2. Перейдите в Application → Local Storage
3. Скопируйте значение ключа `inteca_blog_posts`
4. Сохраните в текстовый файл

Для восстановления:
1. Вставьте JSON обратно в localStorage
2. Обновите страницу

## Техническая информация

### Структура поста:

```typescript
{
  id: string           // Уникальный ID
  title: string        // Заголовок
  excerpt: string      // Краткое описание
  content: string      // HTML контент
  author: string       // Автор
  date: string         // Дата (YYYY-MM-DD)
  tags: string[]       // Массив тегов
  image: string        // URL изображения
  featured: boolean    // Избранный?
}
```

### API Context:

```typescript
const { posts, addPost, updatePost, deletePost, getPost } = useBlog()
```

## Поддержка

При возникновении проблем:
1. Проверьте формат данных
2. Убедитесь, что изображения доступны
3. Проверьте правильность HTML в контенте
4. При необходимости очистите localStorage и начните заново
